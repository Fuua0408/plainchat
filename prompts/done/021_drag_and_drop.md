# 021: ファイル添付のドラッグ&ドロップ対応

## 1. 目的
チャット画面へファイルをドラッグ&ドロップして添付できるようにする。018で作った
添付フロー(画像/テキストファイル・合算4・チップ・送信)への入力経路をD&Dで追加する、
フロントエンドのみのタスク。

## 2. 対象
Phase 3 ②「ファイル添付」のUX追加(D&D)。DECISIONS.md「ファイル添付のドラッグ&ドロップ対応」
(2026-07-05)に従う。

## 3. 前提・参照
- 013〜020完了済み。フロントの添付基盤(public/js/app.js):
  - selectedAttachments が {type:'image'|'file', ...} で画像・ファイルを統合管理し、
    上限は画像+ファイルの合計4(合算)。ボタン選択→分類→上限チェック→プレビュー追加→
    送信時にまとめてアップロード、という経路が既に実装済み(018)
  - チップ生成(createFileChip 等)・プレビュー描画・送信フロー(handleSend)・
    履歴表示は実装済み。D&Dはこれらを作り替えず、入口だけ足して合流させる
  - 画像は uploadImage、テキストファイルは uploadFile を送信時に呼ぶ(既存)
- 対応形式: 画像(image/*)と、テキスト系 .txt/.md/.csv/.json
  (text/plain, text/markdown, text/csv, application/json)。それ以外は非対応
- 技術方針: ビルドなしの素のHTML/CSS/JS、CDNのみ。新規依存を追加しない
- prompts/ 配下は指示書であり、移動以外の編集はしない

## 4. 要件

### ドロップ領域とデフォルト抑止
- チャットペイン(メッセージ表示エリア+入力フォームを含む領域)をドロップ対象にする
- 領域上での dragover は preventDefault してドロップを許可する。
  ブラウザ既定のファイルを開く挙動を、この領域で抑止する
  (領域外への誤ドロップでページ遷移しないよう配慮)

### 視覚フィードバック
- ファイルがドロップ領域上にある間、領域をハイライト(オーバーレイまたはボーダー等)する
- dragenter/dragleave が子要素をまたぐ際のちらつきを防ぐ(dragenterカウンタ、
  または relatedTarget 判定などで、激しい点滅を起こさない)
- ドロップ完了・領域外退出でハイライトを解除する

### ドロップ処理(既存フローへ合流)
- drop 時、dataTransfer.files の各ファイルを type/拡張子で振り分ける:
  - image/* → 既存の「画像を追加する経路」へ渡す
  - .txt/.md/.csv/.json(上記MIME) → 既存の「テキストファイルを追加する経路」へ渡す
  - いずれでもない → 追加せず、非対応である旨をまとめて通知(既存の通知手段に合わせる)
- 追加は既存の add 経路を再利用し、合算4の上限・重複・プレビュー追加・チップ生成を
  既存ロジックに委ねる。上限超過分は既存挙動どおり無視+通知
- ボタン選択で既に追加済みの添付と混在してもよい(合算4を跨いで正しくカウントされること)
- 受信中(送信ストリーミング中)はドロップを受け付けない(添付ボタン無効と同じ扱い)

## 5. やらないこと
- クリップボード貼り付け(paste)対応(将来候補のまま)
- バックエンドの変更
- 添付管理・合算4・アップロード・チップ・送信フロー・履歴表示ロジックの作り替え
  (D&Dは入口の追加のみ。既存経路へ合流させる)
- 非対応形式のアップロード試行(入口で弾く)
- チップのデザイン/視認性の変更(別タスク)
- ドラッグ&ドロップによる並べ替え等、添付以外のD&D
- .env の変更

## 6. 完了条件
ブラウザ(http://localhost:18091)で確認:
1. ファイルをチャットペインにドラッグ → 領域がハイライトされ、外へ出す/ドロップで解除される
2. テキストファイルをドロップ → チップがプレビューに追加され、送信すると内容に即した応答が返る
   (既存の送信フローへ合流していることの確認)
3. 画像をドロップ → サムネイルが追加され、送信できる
4. 画像+ファイルを一度に複数ドロップ → 合算4まで追加され、超過分は無視+通知。
   ボタン選択済みの添付と混在しても合計4で正しく頭打ちになる
5. 非対応形式(例: .pdf)をドロップ → 追加されず通知され、アップロードも走らない
6. ドロップしてもブラウザがファイルを開かない(ページ遷移しない)
7. 送信ストリーミング中はドロップが受け付けられない
8. 子要素をまたいでドラッグしてもハイライトが激しく点滅しない
9. コンソールに未処理エラーがない
10. テストで作成した会話・添付を削除して完了
